# Introduction to PowerShell

## What is PowerShell?
PowerShell is a task-based command-line shell and scripting language
built on the .NET framework. It is used for automation and system
administration.

---

## Key Features
- Object-based (not text-based)
- Built on .NET
- Powerful automation tool
- Used for Windows, Azure, and servers
- Supports scripting and cmdlets

---

## Cmdlets
- Cmdlets follow **Verb-Noun** format
  - Example: Get-Service, Get-Process
- Cmdlets return objects

---

## Important Commands Learned

| Command | Description |
|------|------------|
| Get-Host | Shows PowerShell host details |
| Get-Command | Lists available cmdlets |
| Get-Help | Displays help for a cmdlet |
| Get-Service | Lists Windows services |
| Get-Member | Shows object properties & methods |

---

## PowerShell Help System
- `Get-Help Get-Service`
- `Get-Help Get-Service -Examples`
- `Get-Help Get-Service -Online`

---

## Key Takeaways
- PowerShell works with objects
- Cmdlets are easy to understand and remember
- Help system is very powerful
- Used for automation in real-world scenarios
